Project Idea

AI-Based Real-Time Vehicle & Human Detection System

What It Does

A real-time AI system that detects vehicles and humans from a live camera feed and estimates their distance to enable smart driver assistance.

How It Works
  • Detects 2-wheelers, 4-wheelers, 6-wheelers & humans
  • Estimates distance of each detected object
  • Draws bounding boxes with class labels
  • Streams live output via Flask to browser
System Behavior
  • Above 300m — Alert: "Vehicle detected ahead"
  • 200m – 300m — Tracks & updates distance live
  • Below 200m — Activates slow automatic braking
Purpose
  • Accident prevention
  • Smart driver assistance
  • Road safety improvement
